◆ putToTrash()

void NotesManager::putToTrash ( unsigned int  id)

Mettre une note à la corbeille.

Cette méthode permet de mettre une note qui est supprimée à la corbeille. Si nous avons le temps cette méthode verifiera que la note en question n'est pas dans un couple de la relation référence. Le cas échéant elle serait alors archivée et non mise à la corbeille tant qu'elle est dans ce couple. Si la deuxième note qui forme le couple avec la première venait elle aussi à être supprimée elle serait alors toutes les deux mises à la corbeille.

Parameters
idId de la note que l'on souhaite mettre à la corbeille
